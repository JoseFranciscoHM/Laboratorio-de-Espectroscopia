# Laboratorio-de-Espectroscopia
El software requerido para el diseño del laboratorio de espectroscopia se basa en en los codigos para el manejo de los distintos instrumentos a su vez tambien se incluyen las plantillas para el diseño del rotor.

El sistema de Laboratorio de reflectancia version 1, se divide en distintas secciones para el manejo del monocromador, el manejo del LockIn y para realizar medidas de 
reflectancia y reflectancia diferencial. 

Tambien se anexan los documentos utilizados para el control del analizador rotante.
